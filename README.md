# django-signup-and-login

 Django Signup and Login
 
Functionality
                           
                           
  >Signup
           
           
    
    via username , First Name, Last Name, Email & password
          
          
          
  
  
  >Signin
    
    
    
    
    via username & password
    
    
    
    
    
   >Apply Migrations
   
   
   
   
    python manage.py makemigrations
    python manage.py migrate
    
    
    
    
    
   > Running
   
      
      
      Just run this command:
      
          python manage.py runserver
